Project Overview and Deliverables

The project involves developing a static website for a restaurant facing operational challenges due to high customer volume, leading to order mix-ups and booking issues.
Deliverables include a fully functional static website hosted on AWS and a PowerPoint presentation detailing the restaurant's overview, challenges, AWS services, cost analysis, and migration benefits.

Static Website Development

The website should be user-friendly, featuring a booking form, order submission form, and confirmation page.
Images can be stored in Amazon S3, and the design should use minimal colors for readability while incorporating engaging visuals like photos and videos.

AWS Services Presentation

The restaurant, experiencing inefficiencies with manual systems, needs a centralized solution to manage bookings and orders.
Recommended AWS services include Amazon S3 for hosting the website and images.
The presentation should highlight the advantages of AWS, emphasizing improved efficiency, reduced errors, and scalability.

Presentation Guidelines

Ensure conciseness and clarity in slides, using notes for additional details.
Incorporate engaging visuals while maintaining text visibility through minimal color use.
