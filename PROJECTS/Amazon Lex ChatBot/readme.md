### AWS Lex Interactive Ec2QuizBuddyChatbot Project

### Introduction

In this project, we designed and built an interactive chatbot using Amazon Lex as part of the AWS re/Start Special Projects programme (Praesignis). Our goal was to explore how conversational AI works in practice and to create a chatbot that feels natural, useful, and engaging for users.

The project was completed in two phases:

   * Part 1: Creating a basic chatbot to understand the fundamentals of Amazon Lex
   * Part 2 (Challenge): Expanding the chatbot into a quiz-based learning tool using conditional logic.

### Project Overview

- We built an Amazon Lex chatbot that uses Natural Language Processing (NLP) to understand user input and respond in a structured and meaningful way. The chatbot was designed to guide users through both informational and quiz-style conversations.

- By the end of the project, we had a working solution that demonstrates how chatbots can be used as simple, interactive learning tools, especially in cloud education environments.
  
--- 

### What We Set Out to Achieve

Through this project, we aimed to:

- Understand how Amazon Lex handles intents, utterances, and responses
- Design clear and logical conversation flows
- Build an interactive quiz that responds differently based on user input
- Test and improve the chatbot through hands-on experimentation
