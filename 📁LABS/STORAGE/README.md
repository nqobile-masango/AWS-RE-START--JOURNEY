### Amazon S3 Lab
- This lab explores the creation of an S3 bucket uploading objects and configuring permissions to make the objects accessible through a browser.

### Tasks
### Created and configured an S3 bucket 
- Set up a new S3 bucket with a unique name and configured its settings for access management
### Uploaded objects
- Uploaded an index html, which make up my website, to the S3 bucket via AWS Management Console
### Set permission and access policies
- Blocked public access at the bucket level but configured the bucket policy to allow public read access to the objects so the website could be viewed.
### List the contents of the S3 bucket using the AWS CLI
- Used the AWS CLI command aws to verify and list all files stored in the bucket.
### Why it matters
- Amazon S3 bucket is widely used to store static websites, backups and large files in the cloud.
- Learning how to use Amazon S3 is important because it widley used for most cloud projects
### Challenges
The challenge I faced was listing the AWS contents using the AWS CLI as it was my first time using it
### Takeaways
- Learned how to create s3 buckets and how to upload objects
- Learned how to connect to the AWS CLI and use some commands.
- Learned the bucket policies
